## Rules Must Follow

1. Whenever posting comments, creating items or pushing changes to a public platform, always ask before doing so. This includes git actions, Linear, Slack, Email etc.
2. If the last commit is not pushed, ask user if the current changes should be amended to the last commit or seperated.
3. Always ask user if local plans or docs should be included in commits. Never commit them without asking.
4. Always confirm git branch before make changes.
